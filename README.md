# Main Feature

<ul>
  <li>Login</li>
  <li>Create account</li>
  <li>Read article</li>
  <li>Chat with doctor</li>
  <li>Make appointment</li>
  <li>Edit profile</li>
  <li>Logout</li>
</ul>
