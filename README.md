# Android-Alodokter-K2
Android team kelompok 2 Alodokter bootcamp


A mobile app for online doctor consulatation 

# Main Feature

<ul>
  <li>Login</li>
  <li>Create account</li>
  <li>Read article</li>
  <li>Chat with doctor</li>
  <li>Make appointment</li>
  <li>Edit profile</li>
  <li>Logout</li>
</ul>

# Tech Stack

<ul>
  <li>100% Kotlin</li>
  <li>AndroidX</li>
  <li>Jetpack Library</li>
  <li>RX Java</li>
  <li>Glide</li>
  <li>Retrofit</li>
</ul>
